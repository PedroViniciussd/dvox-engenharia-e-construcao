# D/vox Construções Premium

Projeto Next.js completo com:
- Home moderna com animações, parallax e grafismos de construção
- Componentes reutilizáveis
- Páginas estratégicas SEO
- Sitemap e robots
- Favicon, icon-512 e manifest.json
- Schema.org LocalBusiness, WebSite, Service e FAQPage

## Rodar localmente
npm install
npm run dev

## Deploy
Suba no GitHub e conecte na Vercel.

## Após o deploy
No Google Search Console, envie:
sitemap.xml
